### **Akınsoft Wolvox 8 AsBirChat**

AsBirChat **"Birikim Bilgisayar"** tarafından AKINSOFT Wolvox8 ERP uygulaması için yazılmış **ücretsiz** bir sohbet eklentisidir.

**Özellikleri**

* Tüm kullanıcılar ile haberleşme
* Smile Gönderimi
* Link Preview özelliği
* Dosya Gönderimi
